API de Determinação Judicial

Endpoints:

  • GET /determinacaoJudicial/: Lista de todos as determinações judiciais
  • GET /determinacaoJudicial/<codigo>: Recupera uma determinacao judicial pelo código
  • GET /determinacaoJudicial/dependente/<codigo>: Recupera ums determinacao judicial pelo código do dependente
  • GET /determinacaoJudicial/servidor/<matricula>: Recupera uma determinacao judicial pela matrícula do servidor
  • POST /determinacaoJudicial/: Inclui nova determinacao judicial
  • PUT /determinacaoJudicial/<codigo>: Atualiza uma determinacao judicial pelo código
